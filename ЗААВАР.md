# 🤖 Өдөр тутмын AI мэдээний Telegram бот — Суулгах заавар

Энэ бот өдөр бүр өглөө **9:00 цагт** (Улаанбаатарын цагаар) танд:

- 🗞 Хамгийн сүүлийн үеийн AI мэдээг **монголоор** товчилж
- 🎬 Тухайн мэдээнд суурилсан **бичлэгийн 3 санаа** (TikTok/Reels/YouTube)

илгээнэ. Сервер худалдаж авах шаардлагагүй — GitHub Actions (үнэгүй) дээр ажиллана.

---

## Алхам 1: Gemini API түлхүүр авах (үнэгүй, 2 минут)

1. https://aistudio.google.com/apikey руу орно (Google акаунтаараа нэвтэрнэ)
2. **"Create API key"** товч дарна
3. Гарч ирсэн түлхүүрийг хуулж авна (жишээ: `AIzaSy...`)

> Gemini-ийн үнэгүй эрх өдөрт олон зуун хүсэлт зөвшөөрдөг тул өдөрт 1 удаа
> ажилладаг энэ ботод бүрэн хангалттай.

## Алхам 2: Chat ID-гаа олох (1 минут)

1. Telegram дээр **өөрийн бот руугаа** `/start` гэж бичиж илгээнэ
2. Дараах холбоосыг browser дээрээ нээнэ:

   ```
   https://api.telegram.org/bot8769929384:AAFsqckEe2RlZMqaKOqyiTbR2CQAQ-Y02RU/getUpdates
   ```

3. Гарч ирсэн текстээс `"chat":{"id":1234567890` гэсэн хэсгийг олж,
   тоог нь хуулж авна — энэ таны **Chat ID**

> Хэрэв `"result":[]` гэж хоосон гарвал бот руугаа мессеж илгээгээгүй байна —
> эхлээд `/start` гэж бичээд дахин нээгээрэй.

## Алхам 3: GitHub repo үүсгэх (5 минут)

1. https://github.com дээр акаунт үүсгэнэ (байгаа бол нэвтэрнэ)
2. https://github.com/new дээр шинэ repo үүсгэнэ:
   - Repository name: `ai-news-bot`
   - **Private** гэдгийг сонгоно ⚠️
   - **"Create repository"** дарна
3. Файлуудаа оруулна:
   - **"uploading an existing file"** линк дээр дарж `ai_news_bot.py` болон
     `requirements.txt` хоёрыг чирж оруулаад **"Commit changes"** дарна
   - Дараа нь **"Add file" → "Create new file"** дарж, файлын нэрэнд яг:

     ```
     .github/workflows/daily-ai-news.yml
     ```

     гэж бичээд, `daily-ai-news.yml` файлын агуулгыг хуулж буулгаад
     **"Commit changes"** дарна

## Алхам 4: Нууц түлхүүрүүдээ хадгалах (3 минут)

Repo дотроо: **Settings → Secrets and variables → Actions → New repository secret**

Дараах 3 secret-ийг нэг нэгээр нь үүсгэнэ:

| Name | Value |
|------|-------|
| `TELEGRAM_BOT_TOKEN` | `8769929384:AAFsqckEe2RlZMqaKOqyiTbR2CQAQ-Y02RU` |
| `TELEGRAM_CHAT_ID` | Алхам 2-т олсон тоо |
| `GEMINI_API_KEY` | Алхам 1-т авсан түлхүүр |

## Алхам 5: Туршиж үзэх 🚀

1. Repo-ийнхоо **"Actions"** таб руу орно
   (анх удаа бол "I understand... enable them" товч дарна)
2. Зүүн талаас **"Daily AI News to Telegram"** сонгоно
3. Баруун талд **"Run workflow" → "Run workflow"** дарна
4. 1-2 минутын дараа Telegram-даа мэдээгээ хүлээж авна! 🎉

Ажиллахгүй бол Actions таб дээрх ажлын log-ийг нээж алдааг харна уу
(ихэвчлэн secret-ийн нэр буруу бичигдсэнээс болдог).

## Түгээмэл асуултууд

**Цагийг өөрчлөх бол?** — `daily-ai-news.yml` доторх `cron: "0 1 * * *"`-ийг
засна. UTC цагаар бичигддэг (УБ цаг − 8). Жишээ: орой 20:00 УБ = `"0 12 * * *"`.

**Мэдээний эх сурвалж нэмэх бол?** — `ai_news_bot.py` доторх `FEEDS`
жагсаалтад RSS хаяг нэмнэ.

**Бичлэгийн санааны тоог өөрчлөх бол?** — `ai_news_bot.py` доторх prompt
хэсгийг (Санаа 1, 2, 3...) засна.

**GitHub Actions-ийн cron заримдаа 5-15 минут хоцордог** — энэ хэвийн зүйл,
GitHub-ийн ачааллаас шалтгаална.

⚠️ **Аюулгүй байдал:** Bot token-оо хэнтэй ч хуваалцаж болохгүй. Хэрэв
алдагдсан гэж бодвол Telegram дээр @BotFather → `/mybots` → ботоо сонгоод →
API Token → **Revoke** хийж шинэ токен аваад, GitHub secret-ээ шинэчлээрэй.
